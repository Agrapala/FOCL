FLoBC — Federated Learning on Blockchain
Privacy-Preserving FL Framework for Blockchain Networks
PaperAbuzied et al., Cluster Computing, 2024
DOI10.1007/s10586-024-04273-1
AuthorsYoussif Abuzied, Mohamed Ghanem, Fadi Dawoud, Habiba Gamal, Eslam Soliman, Hossam Sharara, Tamer ElBatt
DatasetChest X-ray pneumonia — 4 Sri Lankan hospitals (Galle, Colombo, Kandy, Jaffna)
Research Objectives
Obj 1 — FL platform integrated with blockchain across ≥4 institutions
Obj 2 — ≥5% accuracy improvement per institution through federated training
Obj 3 — Blockchain integrity verification + malicious update prevention
Obj 4 — Fully functional prototype with Ed25519 credential management
Run Order
python demo.py # quick sanity check
python train_local_nodes.py # stage 1
python run_pneumonia.py # stage 2
python evaluate_objectives.py # stage 3
python run_all.py # OR run everything at once
